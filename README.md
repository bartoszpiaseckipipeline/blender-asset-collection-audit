# Description:
# Tool for validating Blender asset collection structure and
# generating a readable text report for pipeline verification.
#
# Use Case:
# Designed for game development workflows (e.g. Unreal Engine)
# to quickly inspect hierarchy, naming consistency and asset setup.
#
# Features:
# - Active collection export
# - Hierarchy preservation
# - Metadata (scene, file, timestamp)
#
